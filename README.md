This Movie Recommendation System uses a hybrid movie recommendation system using the MovieLens dataset.

Methods used:
2 - K-Means clustering for grouping users
1 - Gradient Boosting and Neural Networks for rating prediction

Evaluation metrics:
1 - RMSE
2 - Precision
3 - Recall
